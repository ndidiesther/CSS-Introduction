What I Learnt so Far

CSS stands for Cascading Styles Sheet. It is used for website layout in design and used alongside HTML. CSS is a styling language. Along with HTML, CSS is fundamental to web design. Without it, websites would still be plain text on white backgrounds. There are three major ways of applying CSS to your HTML and these includes

  • Inline Styling: Here you add your CSS styling in the opening tag of your HTML Elements as a style attribute.
  • Internal styling: An internal CSS is used to define a style for a single HTML page. It is defined in the 'head' section of an HTML page, within a 'style' element.
  • External Styling: An external file is used to define the styling in this method for any HTML page, This method of styling can be used by more than one HTML page.

Also, there are various CSS Selectors which includes:

  • Universal CSS Selector: Selects every Element in the HTML file
  • CSS Element Selector: CSS Element Selector is also known as a Type selector.
  • CSS ID Selector: CSS ID selector helps the developer to match the ID created by the developer to its styling content.
  • CSS Class Selector: The CSS Class selector is one of the most helpful selectors of all the selectors.
  • CSS Class Attribute Selector: The CSS Attribute selector styles content according to the attribute and the attribute value mentioned in the square brackets.

I also learnt about some of the most fundamental concepts of CSS; the cascade, specificity, and inheritance, which control how CSS is applied to HTML and how conflicts between style declarations are resolved. Finally, I also learnt about the box-model and major CSS properties which include comments, colors, background properties, positions, shorthand property, border, margin, padding, CSS height, width and maximum width, outline, outline style, text alignment, and text direction, CSS text decoration, and lists.

Importance of CSS

CSS allowed several innovations to webpage layout, such as the ability to:

  • Specify fonts other than the default for the browser
  • Specify color and size of text and links
  • Apply colors to backgrounds
  • Contain webpage elements in boxes and float those boxes to specific positions on the page

Padding, Margins and Borders

Paddings

It is the space between the element and the related content inside it. It determines how elements look and sit within a container. It also shows the container background around the element in it. Padding can be affected by background colors as it clears the area around the content. To create the gap, it either grows the element size or shrinks the content inside. By default, the size of the element increases.

Margins

It is the space around an element. Margins are used to move an element up or down on a page as well as left or right. Margin is completely transparent, and it does not have any background color. It clears the area around the element. Each side of the element has a margin size you can change individually. In creating the gap, the margin pushes adjacent elements away

Borders

The CSS border is a shorthand property used to set the border on an element. The CSS border properties are use to specify the style, color and size of the border of an element.
